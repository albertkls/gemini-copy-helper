# Gemini Copy Helper 🤖

这是一个专为 **Trae** 和 **VS Code** 开发的 AI 辅助插件。它可以一键提取代码上下文和报错信息，并生成适合发给 Gemini/ChatGPT 的 Prompt。

## ✨ 核心功能 (Features)

插件会根据你当前的文件类型，智能决定抓取策略：

1.  **📓 Jupyter Notebook (.ipynb) [全自动]**
    * 无需任何操作！插件会自动遍历所有单元格，精准抓取**红色的报错信息 (Traceback)** 和相关代码。

2.  **🐍 普通代码文件 (.py/.js 等) [智能判定]**
    * **手动选中优先**：如果你在终端(Terminal)选中了一段报错日志，插件会优先复制选中的内容（解决运行时报错）。
    * **静态检测兜底**：如果你没选中任何字，插件会自动抓取编辑器里画**红波浪线**的语法错误。

## ⌨️ 快捷键 (Shortcuts)

为了极速体验，建议养成使用快捷键的习惯：

* **Mac:** `Cmd` + `Alt` + `C`
* **Windows:** `Ctrl` + `Alt` + `C`

## 📖 使用指南 (Usage Guide)

### 场景 A：在 Notebook 中遇到报错
1.  代码运行出错，单元格下方出现红色 Traceback。
2.  **直接按下快捷键**。
3.  等待提示 "✅ 已复制 (来源: Notebook Auto-Detect)"。
4.  去 AI 网页版粘贴。

### 场景 B：在运行 .py 脚本时终端报错
1.  终端输出一堆报错信息。
2.  **用鼠标选中**终端里那段关键的报错文字。
3.  **按下快捷键**。
4.  等待提示 "✅ 已复制 (来源: User Selection)"。

### 场景 C：代码还没跑，编辑器就飘红
1.  不用运行，**直接按下快捷键**。
2.  插件会自动提取红线处的错误详情。
