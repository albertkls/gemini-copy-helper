# Gemini Copy Helper

这是一个专为 Trae 和 VS Code 开发的辅助插件。

## 功能 (Features)

当你在写代码遇到报错时：
1. **自动提取**：插件会自动提取当前文件的**报错信息**和**代码上下文**。
2. **智能拼接**：生成一段包含错误描述和相关代码的 Prompt。
3. **一键复制**：直接复制到剪贴板，你只需去 Gemini/ChatGPT 粘贴即可。

## 快捷键 (Shortcuts) 🔥 新增

为了提高效率，你可以直接使用快捷键触发：

* **Mac:** `Cmd` + `Alt` + `C`
* **Windows:** `Ctrl` + `Alt` + `C`

## 使用方法 (Usage)

1. 打开一个有报错的代码文件。
2. 使用快捷键，或者按下 `Cmd+Shift+P` 输入 `Ask Gemini`。
3. 等待右下角提示 "✅ 已复制"。
4. 去 AI 网页版粘贴提问。
