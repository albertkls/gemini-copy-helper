# Gemini Copy Helper

这是一个专为 Trae 和 VS Code 开发的辅助插件。

## 功能 (Features)

当你在写代码遇到报错时：
1. 使用命令 `Ask Gemini: Copy Error & Code`。
2. 插件会自动提取当前文件的**报错信息**和**代码上下文**。
3. 自动生成一段 Prompt 并复制到剪贴板。
4. 你可以直接去 Gemini 网页版粘贴提问。

## 使用方法 (Usage)

1. 打开一个有报错的代码文件。
2. 按下 `Cmd+Shift+P` (或 Ctrl+Shift+P)。
3. 输入 `Ask Gemini` 并回车。
